# Git-harjoitukset

Tämä projekti sisältää Gitin harjoitustehtäviä.
Harjoituksissa on käytetty Gitin perustoimintoja kuten:
- Repositorion luominen
- Branchien käyttö
- Mergaukset ja konfliktien ratkaisu
- GitHubin etärepositorion käyttö
- Git stash ja tagit

Projekti on tehty Visual Studio Codessa.
